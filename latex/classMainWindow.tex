\doxysubsection{Detailed Description}
The main window for the whole application. 

This window is designed with a UI file. The purpose of this is to setup the main structure of the application. It\textquotesingle{}s responsible for instansiating the other window respectevly. It\textquotesingle{}s structured such that withing the central widget there is a frame, depening on which buttons the user presses the cooresponding object is created.
