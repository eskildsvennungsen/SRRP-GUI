# SRRP - GUI
A simple graphical user interface for controlling and monitoring the SRRP system. 

## Requirements:
- Qt6
- Cmake

#### Install:
```console 
git clone https://github.com/eskildsvennungsen/SRRP-GUI
cd SRRP-GUI
sudo pacman -S - < qtreq.txt
```
#### Build:
```console
cmake -S . -B /path/to/build
cd /path/to/build
make && ./SRRP-GUI
```
